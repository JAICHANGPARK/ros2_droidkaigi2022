# ros2_droidkaigi2022
 
- FlutterをROS(ロボティクス)と一緒使ったらマジですごかった話

## How to build package

```
> colcon build --symlink-install
> source ./install/setup.bash
```

## How to run 

```
> ros2 launch ros2_droidkaigi2022 ros2_droidkaigi2022.launch.py
```
